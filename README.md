# Student Score Prediction

A beginner Machine Learning project that predicts student exam scores based on study hours using Linear Regression.

## Tech Stack

- Python
- Pandas
- Matplotlib
- Scikit-Learn

## Workflow

1. Load dataset
2. Visualize data
3. Train-test split
4. Train Linear Regression model
5. Evaluate model
6. Predict score

## Algorithm

Linear Regression
